# typescript-clean-arch-template

## General

Install packages

```bash
$ yarn install
```

## Develop

To run all development env

```bash
$ yarn dev:app
```

To run app with watch

```bash
$ yarn dev:app-watch
```

To run development infra with docker compose

```bash
$ yarn dev:infra
```

To run type check

```bash
$ yarn type-check
```

## Tests

To run tests

```bash
$ yarn test
```

To run tests with watch

```bash
$ yarn test:watch
```

To run tests with verbose

```bash
$ yarn test:verbose
```

To run tests to staged context

```bash
$ yarn test:staged
```

To run tests in ci

```bash
$ yarn test:ci
```

## Run transpiled code

```bash
$ yarn start
```
